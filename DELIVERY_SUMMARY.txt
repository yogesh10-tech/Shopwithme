💾 LOCAL STORAGE STRUCTURE
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Cache Storage Key: 'meropasal_barcode_cache'
  • Format: JSON object with version
  • Checksum: SHA-based integrity verification
  • Content: All product barcodes
  • Last Sync: Timestamp of last sync
  • Status: Synced vs unsynced tracking

Sync Queue Key: 'meropasal_barcode_sync'
  • Format: Array of pending operations
  • Storage: Operations queued offline
  • Retry: Auto-retries when online
  • History: Each has timestamp & unique ID
